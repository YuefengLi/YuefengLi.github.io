梯子服务

提供 Shadowsocks 翻墙服务，也就是 SS。
提供翻墙服务的目的是便于访问谷歌学术等墙外的学术网站，请不要经常挂着，不要乱下载大文件或者看视频。目前的服务器同时在线的客户端最多是 5 个，月流量 3TB。

  1. 安装客户端；
  2. 在电脑上打开给出的二维码图片；
  3. 启动客户端，右击任务栏中的小飞机图标，选择『服务器』-『扫描屏幕上的二维码』；
  4. 自动读取得到 SS 服务器配置，点击确定。
  5. 配置客户端为『启用系统代理』，系统代理模式是 『PAC 模式』。如果全局模式的话就是所有流量都走 SS 了。

FTP 文件服务器

本文件服务器可以在任何有网络的地方访问，访问速度取决于网速和硬盘速度。比较适合的应用场景是实验室两台不同局域网之间的电脑互传文件，或者文件备份，或者一对多的文件分发。
本文件服务器的基本文件夹结构如下：

  • Software：包含各种常用的软件，装机必备。只读。
  • SWAP：可读写。其下按照人名建立文件夹存放个人资料。

本文件服务器搭建在我的实验室电脑上，通过『FileZilla Server』服务自启动。所以一般只要保证我电脑开着就行了。
由于未知原因，无法通过浏览器（Chrome/IE etc.）或者 Windows 自带的资源管理器访问。建议使用免费的 FileZilla 客户端（Win 32位/Win 64位）。
下载此免安装版本后解压打开 FileZilla.exe，输入登录信息如下：

  • 主机：ftp.yuefeng.li
  • 用户名：与实验室 WiFi 密码相同
  • 密码：####
  • 端口：21

点击『连接』之后就可以在右边浏览远程目录。左侧是本地目录。

打印机服务

  • 使用随机光盘安装驱动。
  • 过程中，可能会问你如何连接打印机，忽略之。
  • 在『控制面板』-『打印机和设备』中，删除已经安装的黑色惠普 225-226 打印机（如果有的话）。
  • 左上角『添加打印机』-『我需要的打印机未列出』-『使用 TCP/IP 地址或主机名添加打印机』，按如图配置，端口名称随意：

  • 下一步之后等待即可。如果问你要不要共享打印机，选择不要。
  • 不要在配置完成后『打印测试页』，浪费纸。为了测试打印机是否连接成功，可以打开一个空白的 word 文档打印。
  • 如果和打印机在同一个局域网，可以在安装驱动的时候自动搜索，自动配置。
