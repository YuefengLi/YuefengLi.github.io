对本博客的最后致意

越锋利 4月 19, 2020
  • 分享到微博
  • 分享到 Twitter
  • 分享到 Telegram

这篇文章既是一块墓志铭，也是一块指路牌。

现在是 2020 年的 4 月 19 日，我刚花了两个晚上的时间，把整个博客从 Farbox/Bitcron 迁移到了 Hexo + GitHub Pages，刚刚为这个站点换上了崭新的 material design 的主题，心情还有些沉浸在新鲜的小小的成就感里，不过当我停下来，仔细的思考这个博客的未来，我只能得出一个结论：

本文很可能就是这个博客的最后一篇文章了，即使不是最后一篇，这也一定是倒数第几篇了（n < 5）。

一段简单的编年史

一般来说，墓志铭上应该写点生平经历什么的，稍微回顾一下过去，这个博客大概经历过下面这么几个阶段：

  • 2007~2014，百度空间时期
    • 内容：大约 297 篇文章。最后一篇文章，叫做《对本百度空间的最后致意》；
    • 后来：百度空间关闭了，这些文章都静静躺在我的百度云里；
  • 2014~2019，Farbox/Bitcron 时期
    • 转变：短暂的在 OpenShift 上搭建过 WordPress，后转移到 Farbox；
    • 内容：只有大约 30 篇文章。我的大部分输出，都给了微博和 Evernote；
    • 后来：删减了部分之后，留在了这个博客上；
  • 2020~，Hexo 时期
    • 内容：原则上不再新增文章；

为什么要离开这里？

一方面，互联网的形态是在一直变化着的。当年整个互联网的流量集中在门户网站和少数特别出名的个人站点，普通用户要发表自己的想法，只能选择个人或者公共博客空间，通过搜索引擎拉到流量。而在 web 2.0 的时代，随着微博和抖音等等 SNS 网站的出现，互联网上的个人发声更加容易，通过这些关系网相互交织的，面向 UGC 的网站，个人用户能够更轻松地，更意外地吸引到更多的流量。

另一方面，我写博客的目的，甚至于我自己人生存在的意义，也在一直慢慢地演化。我已经不再是那个年轻的时候，随便钻研一点乱七八糟的东西就想写出来炫耀一下的我了，我有很多维度的轮廓（profile），想展示给不同的人看。豆瓣上的用户能看到我关于文艺作品的想法，知乎上的用户能看到我关于技术问题的见解，朋友圈的好友能看到我个人的生活。从浏览者的角度来讲，如果他抱着明确的目的性，应该也不想花太多时间在一个混杂的个人博客里面去寻找他想要的东西的吧。

当然有一些遗憾

  • 这个站点的主题我觉得是蛮好看的，然而仔细看还是有各种大大小小的问题，但我应该也不会去修了。Github 上这个项目也早已是人去楼空，甚至，往大里说，整个互联网上能搜索到的关于 Hexo 的内容，大部分也是两三年前的……
  • 过去的三百多篇文章，有很多在现在的我看来简直就是羞于公开，但也有几篇相当有趣的，不过好像也没有动力去一一整理他们了。也许等我老了，坐在夕阳下无所事事的时候，可以慢慢咀嚼回味……

我将会去哪里？

所以具体来说，关于未来，我打算在以下几个站点输出：

  • 微博：转发新闻和社会观点；
  • 什么值得买：关于生活和购物的想法；
  • 知乎：所有和技术有关的研究；
  • 豆瓣：文艺作品评论。

这里会变成怎样？

我相信 GitHub Pages 不会倒闭，所以这个站点应该会一直存在。

如果有一些我不知道该放到哪里去好的文章，也许我也会把它放在这里。什么叫做「不知道该放到哪里」的文章呢？我目前能想到的，就只有讲爬梯子翻墙的文章了。

回想起来，我年轻的时候还是比较文艺的，我的百度空间，banner 里写着的是一句拉丁文，意思是「时间盲目，人类愚蠢」。这句话好像蛮适合刻在墓志铭上的。
